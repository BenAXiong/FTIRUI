Projects overview (legacy)

The modern dashboard is disabled because DASHBOARD_V2_ENABLED is set to false. Enable the flag (or append ?dev=true) to restore the new explorer + filters. This legacy layout keeps the core entry points available so users can still reach their canvases.

Open a canvas

Jump into the workspace editor, then use the browser sidebar to pick a migrated canvas.

Go to workspace

Run migration

Use python apps/ftirui/manage.py migrate_sessions to move saved sessions into the new Projects/Folders structure.

Need a dry-run? Add --dry-run.

Switch to v2

Set DASHBOARD_V2_ENABLED=true (or append ?dev=true) to preview the new explorer, filters, and list/gallery views.

Remember to flip the flag only after QA signs off.
